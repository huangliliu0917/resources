接口协议：

接口安全验证，将所有请求参数和appSecret按升序，将升序后的values进行md5 utf-8 加密

参数名：sign （服务端验证签名是否正确）

appSecret = "99d0da1165a8d240b29af3f418b8d105"

明文参数中不需要带appSecret

token:身份验证，用户登陆后生成的标识，用来验证用户登陆状态

    登录、注册、初始化自动登陆等接口，都将重新生成新token，客户端必须对token进行处理操作。(为保证同一帐号，在不同设备登陆，只要有一个设备进行了登陆，其他设备自动下线)

operation：

    1、huobanmall2015AD   （android）

    2、huobanmall2015AP   （ios）

接口输出固定格式

Json格式：

{

    resultCode: 1,

    description: "成功",

    status: 1,

    tip: "操作成功",

    resultData: {},

    versionClaim: 0

}

 

参数说明：

resultCode:     系统状态返回：1，成功;0，失败

description:    成功/失败描述

status:         逻辑状态返回 ：1成功,0 失败

tip:            逻辑状态描述

resultData:     业务数据

versionClaim:   此接口是否有版本要求0没有1有,(2 token过期，需要重新登陆,待定)
